+ $|TMin| = |TMax| + 1$
+ $UMax_w = 2TMax_w + 1$
+ -1 和 $UMax$ 有相同的位表示——一个全为 1 的串
+ 0 在两种表示方法中都是一个全 0 的串
+ 声明变量：
	+ uint16-t : 16 位无符号变量
	+ int32_t : 32 位有符号变量

# 1.2.1 有符号数和无符号数转换
##### 1. 补码转换为无符号数:
对满足 $TMin \le x \le TMax$ 的 $x$ 有：
$$T2U_w(x) = 
\begin{cases}
x + 2^w& x < 0 \\
x& x \ge 0
\end{cases}
$$
##### 2. 无符号数转换为补码
对满足 $0 \le u \le UMax_w$ 的 $u$ :
$$U2T_w(x) = 
\begin{cases}
u & u \le TMax_w\\
u - 2^w &u > TMax_w
\end{cases}
$$
