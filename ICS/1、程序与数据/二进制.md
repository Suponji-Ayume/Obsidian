### Bit-Level Manu
1. And : A&B=1 when both A = 1 and B = 1
2. Or : A | B = 1 when either A = 1 or B = 1
3. Not : ~A = 1 when A = 0
4. XOR : A ^ B = 1 when A = 1 , B = 0 or A = 0 , B  = 1
